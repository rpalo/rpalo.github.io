$PYTHONSTARTUP

If you have the environment variable $PYTHONSTARTUP set to a valid Python file, that file will get run when starting up the Python interpreter.

The -i Flag

In addition to these new superpowers, you can temporarily make any Python script your startup script. This could come in really handy for some interactive debugging. Letâ€™s say youâ€™re working on a project and you have a script that defines some functions:

You can use the -i flag when you run the Python interpreter to use cool_script.py as your startup file instead of your usual one.
